# SoluPos

App Android que actúa como contenedor móvil para sistemas POS web.

## Funcionalidades

- Gestión de múltiples tiendas por URL
- Carga del sistema POS en WebView
- Escaneo de códigos de barras con la cámara
- Impresión nativa desde el sistema web

## Requisitos

- Android 8.0 (API 26) o superior
- Acceso a la red donde esté alojado el sistema POS

## Stack

Kotlin · Jetpack Compose · Room · CameraX · ML Kit · Navigation Compose
